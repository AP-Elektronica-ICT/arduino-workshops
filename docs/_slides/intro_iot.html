Bluetooth

  • 2.4 GHz
  • Range: ±100m
  • Versie
    • Huidig: 4.0(BLE)
    • Opkomend: 5.0
  • Point To Point
  • Niet geschikt voor mesh netwerk
  • Makkelijk te implementeren

ZigBee

  • Sub GHz
  • 2.4 Ghz
  • Range: ±1500m
  • Geschikt voor mesh netwerking
  • Secure communicaties

802.11 bg / WiFi

  • 2.4GHz
  • Dagelijks gebruik
  • Makkelijk te implementeren
  • Energie consumerend
